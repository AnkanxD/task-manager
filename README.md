Task Manager App (Full Stack)

This is a simple full-stack Task Manager application where users can:
- Add tasks
- Mark tasks as completed
- Delete tasks
- View all tasks

Tech Stack:
- Frontend: React.js
- Backend: Node.js + Express.js
- Database: MongoDB
- Deployment: Render

Live App:
https://task-manager-1-i763.onrender.com

API Endpoint:
https://task-manager-120z.onrender.com/tasks

Features:
- Create task
- Update task status
- Delete task
- Fetch tasks from database

Project Structure:
- frontend → React app
- backend → Express server & MongoDB

How to Run Locally:
1. Clone the repo
2. Run backend:
   cd backend
   npm install
   npm start

3. Run frontend:
   cd frontend
   npm install
   npm start
